# Contributing to Tesla-Gov

Welcome to **Tesla-Gov**! We appreciate your interest in contributing to our frictionless governance tool for Stellar communities. 

This repository was created by **@AlAfiz** for the `teslabutt` organization as part of the Stellar Drips Wave program.

## Workflow

1. Fork the repository: `https://github.com/teslabutt/Tesla-Gov`
2. Create a new branch: `git checkout -b feature/my-feature`
3. Make your changes and commit them.
4. Push to your fork and submit a Pull Request.

## Drips Wave Sprint Issues

We have outlined several tasks for Drips Wave contributors. Please claim an issue before starting work!

- **Trivial (100pts):** Update the UI to show a "Closed" badge on expired proposals.
- **Medium (150pts):** Add a Fastify endpoint to fetch a user's entire voting history.
- **High (200pts):** Implement token-weighted voting in the Rust contract (1 token = 1 vote).

Please ensure your code passes standard `npm run lint` and `npm run build` checks before requesting a review.
